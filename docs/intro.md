# Johnny-Five. Co to jest

*Johnny-Five*, *Node SerialPort* oraz *NodeBots* w telegraficznym skrócie.

> **Johnny-Five** (<http://johnny-five.io>) is an open source robotics library for Node.JS. – *Kassandra Perch*, *Learning JavaScript Robotics*

> Johnny-Five was built on top of **node-serialport** and allows us to write JavaScript applications that communicate with different microcontrollers using different types of connection. – *Kassandra Perch*, *Learning JavaScript Robotics*

> A **NodeBot** is any piece of hardware that is
controlled using JavaScript and/or Node. JS.
This can encompass a wide variety of projects;
there are hundreds of ways to create a NodeBot. — *Kassandra Perch*, *Learning JavaScript Robotics*
